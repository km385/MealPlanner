# Meal Planner Application

A web application for managing recipes and meal planning built with Spring Boot and Vue.js.

## Technologies

### Backend
- Java 23
- Spring Boot 3.3
- Spring Security with JWT
- MySQL Database
- MapStruct
- Lombok

### Frontend  
- Vue.js 3
- Vite
- Tailwind CSS
- Axios
- Pinia
- Vue Router

## Getting Started

### Prerequisites
- Java 23
- Node.js
- MySQL

### Startup
```sh
./mvnw spring-boot:run
```
```sh
npm install
```
```sh
npm run dev
```
